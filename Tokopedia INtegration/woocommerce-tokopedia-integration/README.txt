== Description ==

Tokopedia Integration for WooCommerce allows user to sync their WooCommerce store and its products to the Tokopedia Marketplace. This plugin can also be used to manage the Tokopedia orders as well directly from WooCommerce Store.

A few notes about the sections above:

*   Upload Simple Products to Tokopedia Marketplace
*   Upload Variable Products to Tokopedia Marketplace
*   Manage Tokopedia Orders directly on WooCommerce

= Minimum Requirements =

* WordPress 4.7 or greater
* WooCommerce 3.0.0 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater
